# Lock The Tile 🎮

Lock The Tile is a Java Swing desktop game inspired by the classic **Shut the Box** game.

---

## ✨ Features

- 🎲 Dice rolling system
- 🔒 Tile locking mechanism
- 👥 Multiplayer mode
- 🤖 Computer opponent
- 🖥️ Java Swing GUI
- 🏆 Winner detection system

---

## 🛠️ Technologies Used

- Java
- Java Swing
- OOP
- Event Handling
- Exception Handling

---

## ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/your-username/lock-the-tile.git
````

### Compile

```bash
javac *.java
```

### Run

```bash
java LockTheTile
```
